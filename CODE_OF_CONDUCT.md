# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and abusive
behavior are not tolerated. Maintainers may remove, edit, or reject contributions
and comments that violate this code, and may ban contributors for behavior they
deem inappropriate.

Report concerns by opening a confidential issue or contacting a maintainer. The
full text of the Contributor Covenant 2.1 is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
